Refined Notepad

Refined Notepad is a work in progress project that is being programmed in Python. It can be used to create and edit text documents.

See The GitHub Repository

Working on Refined Notepad has been a fun challenge.

I was motivated to begin the Refined Notepad project in Summer 2018 while I was attempting to learn more about the Python programming language. I began to think about how I code an application similar to those that regularly appear in our modern day world. Eventually, I came up with the idea of creating a text editor of sorts in order to enhance my knowledge of the Python programming language. I also thought it would be fun to create something that I could use in my daily life. With these goals in mind I set off on creating a notepad-like text editor in Python and began coding in October.

Working option this project has been a fun challenge as I have learned more about Python and industry standards when creating a software project. I first came up with an idea for my project and drew a diagram as to how it is going to function. I then created a repository for the project and (eventually) I began to code. I have utilized the GitHub flow model when I have been completing this project, using the Issue Tracker and Pull Requests to organize and complete tasks for the project.

Having thorough testing is key for any software project to be truly successful. I have had trouble coming up with ways to test Refined Notepad. As of right now, I have resorted to testing my code's quality by checking it with the PEP8 tool that makes sure the program's code is adhereing to the PEP8 style guide. In terms of testing the actualy program, so far I have only done manual testing of the program to make sure everything looks correct and that all of the program's features are functioning as intended. In the future, I hope to find a way to automate some of the testing process by creating some sort of test suite in order to strengthen the reliability of the software and catch bugs before the project is pushed online.

I will update this page with more information over the coming weeks as I work more on this project.
